## LCP 06. 拿硬币

在数组coins中有n个数，每个数表示一堆硬币包含的硬币数。每次任选一堆，那种其中的一枚或两枚，求拿完所有硬币的最少次数。

### 示例1

> 输入：[4, 2, 1]   
> 输出：4   
> 解释：第一堆至少要拿2次，第二堆至少拿1次，第三堆至少拿1次

### 我的代码

    class Solution:
        def minCount(self, coins: List[int]) -> int:
            res = 0
            for coin in coins:
                if coin % 2 == 0:
                    res += coin // 2
                else:
                    res += coin // 2 + 1
            return res
